1 Machine Learning
------------------

What is the main goal of Machine Learning?

A) To follow predefined rules
B) To learn patterns from data
C) To replace databases
D) To increase hardware speed
✅ Answer: B

Machine Learning is a subset of:

A) Data Science
B) Computer Networks
C) Artificial Intelligence
D) Software Engineering
✅ Answer: C

Which of the following best represents the ML workflow?

A) Code → Output → Debug
B) Data → Model → Prediction → Improvement
C) Input → Compile → Execute
D) Design → Deploy → Test
✅ Answer: B

Which is NOT a real-world application of Machine Learning?

A) Spam filtering
B) Fraud detection
C) Manual data entry
D) Recommendation systems
✅ Answer: C

Model generalization refers to:

A) Memorizing training data
B) Performing well on unseen data
C) Increasing model size
D) Reducing dataset size
✅ Answer: B

9 Bias vs Variance
-----------------

Bias refers to:

A) Error due to complex model
B) Error due to simple assumptions
C) Random noise
D) Hardware failure
✅ Answer: B

Variance refers to:

A) Error due to data scarcity
B) Error due to sensitivity to data
C) Error due to bias
D) Error due to labels
✅ Answer: B

High bias leads to:

A) Overfitting
B) Underfitting
C) High variance
D) High accuracy
✅ Answer: B

High variance leads to:

A) Underfitting
B) Good generalization
C) Overfitting
D) Simple models
✅ Answer: C

The goal is to:

A) Maximize bias
B) Maximize variance
C) Balance bias and variance
D) Remove both
✅ Answer: C

C) Accuracy
D) F1-score
✅ Answer: C

C) Correct positive predictions
D) Total predictions
✅ Answer: C

Recall focuses on:

A) Identifying all actual positives
B) Reducing false positives
C) Increasing accuracy
